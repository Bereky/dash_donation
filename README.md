# Installation

 ### Install server dependencies
 ``` cd dash_donation 
     npm install
 ```
 ### Install client dependencies
 ```
     cd client 
     npm install
     npm run build
 ```
 
 ### Serving the application 
 - Go back to the root folder
  ```
    npm run dev
  ```
  - The app will start listening on port ```5200```
  - Go to ```localhost:5200```
